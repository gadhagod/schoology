### login
Connect to your schoology account.

**Syntax** \
`schoology login [OPTIONS]`

**Options** 
- `--key TEXT`
 - Your API key
 - Required
- `--secret TEXT`
 - Your API secret
 - Requried

**Description** \
Authorize to your schoology account. Credentials are stored in `~/.schoology`. \
Get your API credentials from `https://{your school's schoology URL}/api`.
